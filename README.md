# Gamer's Arena

A sample project built using React 16 and Redux.

## Getting Started

### Prerequisites

```
NodeJS
```

### Installing

```
git clone https://github.com/anmolmahatpurkar/GamersArena.git

cd GamersArena

npm install

npm start
```

Sample data being loaded asynchronously through Firebase.
